There are 3 types of lists:

defination lists

  • which shows terms and definitions

unordered lists

  • which show bullets

ordered lists

  • which show numbers or letters

Above is an example of a definition list.

Unordered lists can have a type attribute

which can be applied to the UL or the LI tag.

The three values used are:

  • type = "square"
  • type = "disc"
  • type = "circle"

Ordered lists can have a type attribute too.

Used along the type attribute they can show:

  1. lower case letters
  2. upper case letters
  3. lower case roman numerals
  4. upper case roman numerals
  5. numbers
